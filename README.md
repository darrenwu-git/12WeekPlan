# 🚀 12 Week Plan (NorthStar System)

這是一個基於《12 週計畫》 (The 12 Week Year) 與 OKR 框架設計的個人管理系統，旨在將長期願景轉化為每週、每日可執行的「行動指標」。

## 🌟 核心理念
這套系統的核心是 **「高頻率執行的行動指標 (Lead Indicators)」**。與其關注結果，不如專注於你每天可以控制的行為。當行為到位，結果自然會跟上。

## 📂 目錄結構與用途
- `202X/`: 存放各年份的執行數據（在此公開版本中為範例結構）。
- `Templates/`: **核心組件**，包含所有板模：
  - `year_okr_template.md`: 年度願景與季度 OKR 規劃。
  - `week_plan_template.md`: 每週行動指標設定。
  - `daily_task_template.md`: 每日 Big 3 任務與反思。
- `README.md`: 系統說明書。

## 🛠️ 如何建立你的 12 週計畫系統？

1. **定義年度願景**：使用 `year_okr_template.md` 規劃你的四大領域（健康、工作、玩樂、愛）。
2. **拆解 12 週循環**：將年度目標拆解為 12 週的具體行動指標。
3. **建立執行目錄**：依照 `年份/月份/Week_XX` 的格式建立資料夾。
4. **每日執行與檢討**：
   - 每天早上寫下 **Big 3** 重點。
   - 每週末進行 **Weekly Review**，計算行動指標的達成率。

## 🐈 數位助理：皮皮 (Pippi)
在我的私人版本中，我使用 OpenClaw 驅動的 AI 助理「皮皮」來輔助：
- **自動排程**：每天 05:30 自動彙整 Google Keep 靈感與 Markdown 任務發送簡報。
- **自動分數統計**：掃描 `week_plan.md` 勾選狀況並統計執行分數。
- **跨平台同步**：將 Telegram 的點子同步至 Google Keep 與 Markdown 檔案。
